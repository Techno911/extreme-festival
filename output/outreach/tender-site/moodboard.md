# МУДБОРД 1: VISUAL STYLE

### Реф 1: Wacken Open Air — официальный сайт
**Источник:** https://www.wacken.com

Тёмный фон #0a0a0a, логотип пламенем, жирный белый шрифт без засечек. Hero — живые фото толпы в пите с факелами, не постановочные, реальный хаос. Огненный акцент как единственный цвет.

**ЧТО БРАТЬ:** Цветовая гамма (чёрный + огненный акцент) + принцип «фото важнее графики» + полная темнота без компромиссов.
**ЧТО НЕ БРАТЬ:** Немецкая перегруженность — слишком много элементов на одном экране.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Wacken — эталон доверия у металлистов 35-45 лет. Если сайт «выглядит как Wacken» — это автоматически «серьёзный фест» без пафоса.

### Реф 2: Download Festival — сайт 2024
**Источник:** https://www.downloadfestival.co.uk

Hero — живое фото хедлайнера на огромной сцене, минимальный оверлей, крупный шрифт с разреженным tracking в левом верхнем углу. Ниже — карточки лайнапа с минимальными рамками. Оранжево-красный градиент акцентных элементов.

**ЧТО БРАТЬ:** Подход к верстке лайнапа — не сетка, а иерархия по шрифту (хедлайнер огромный, остальные меньше). Фото дышит, не задушено плашками.
**ЧТО НЕ БРАТЬ:** Цветовую схему — UK-стиль, нам ближе промышленная грубость.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Лайнап-блок Download — один из лучших на рынке. Визуально понятна иерархия групп без таблиц и звёздочек.

`[Visual]`

### Реф 3: Hellfest — сайт 2023
**Источник:** https://www.hellfest.fr

Адская эстетика, но при этом UX чистый. Hero на полный экран — живое фото толпы ночью с огнями. Навигация — одна строка сверху, без мегаменю. Максимальная атмосфера при минимуме элементов.

**ЧТО БРАТЬ:** Принцип one-image hero + ночная съёмка как основной стиль фото. Один сильный образ вместо пяти слабых.
**ЧТО НЕ БРАТЬ:** Религиозную символику (кресты, пятиконечные звёзды) — нам это не нужно.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Hellfest решил главную проблему метал-сайтов — «страшно, но некомфортно». У них страшно И удобно. Хотим то же для Эстрим Феста.

### Реф 4: Bloodstock Open Air — сайт
**Источник:** https://www.bloodstock.uk.com

Ярко-красный как основной акцент поверх абсолютной темноты. Металлическая текстура в фоне — как засаленный плакат, не цифровая гладкость. Шрифт с засечками для заголовков — отсылка к старым флаерам.

**ЧТО БРАТЬ:** Идею текстурного фона — не просто #000, а с фактурой: царапины, зернистость. Тактильное ощущение реального материала.
**ЧТО НЕ БРАТЬ:** Красный как основной акцент — у нас должен быть свой цвет.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Bloodstock — единственный фест, чей сайт выглядит как старый металлический журнал. Аудитория 35-45 лет считывает это мгновенно: «свои делали».

### Реф 5: Nine Inch Nails — nin.com
**Источник:** https://www.nin.com

Монохромный, брутально минималистичный. Белый текст на чёрном без компромиссов. Никакой графики. Только сильная типографика и атмосферная концертная фотография в одном большом блоке. Уверенность через пустоту.

**ЧТО БРАТЬ:** Принцип «один сильный элемент на экране». Не заполнять каждый пиксель. Когда контент сильный — украшения не нужны.
**ЧТО НЕ БРАТЬ:** Чистую монохромность — нам нужен один акцентный цвет поверх чёрно-белого.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Наша аудитория ценит уверенность. Сайт, который не пытается никому понравиться, уже вызывает уважение.

`[Visual]`

### Реф 6: Godspeed You! Black Emperor — постерный стиль
**Источник:** Behance — поиск «GY!BE poster design» / «metal festival poster zine aesthetic»

Постеры в стиле советских агитплакатов + нойз-эстетика. Зернистость, выцветший вид, текст поверх документальных фото. Грубо, без шлифовки — как распечатка с ксерокса.

**ЧТО БРАТЬ:** Идею «живого документа». Блоки с текстом и фото на сайте могут иметь этот грубый стиль — не всё причёсано одинаково.
**ЧТО НЕ БРАТЬ:** Леворадикальный политический контекст — у GY!BE это неотделимо от эстетики, нам не нужно.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Металлисты 35-45 лет ценят «сделано руками». Этот реф помогает избежать ловушки «красивого корпоративного тёмного сайта».

### Реф 7: Parkway Drive — концертная съёмка
**Источник:** Instagram @parkwaydrive / тег «metal photography» на Behance

Концертные фото в стиле high-contrast monochrome + single colour flare. Дым, свет снизу, прыжки в толпе. Не постановка — момент. Один яркий элемент цвета (вспышка, пламя) на монохромной основе — мощный визуальный удар.

**ЧТО БРАТЬ:** Принцип цветокоррекции для наших живых фото — поднять контраст, убрать тёплые тона, оставить один акцент.
**ЧТО НЕ БРАТЬ:** Это стиль фото, не сайта — не переносить прямо как фон, только как обработку.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Наши фото с Эстрим Феста должны обрабатываться по единому стилю. Без этого рефа дизайнер не поймёт как именно — «просто тёмные» недостаточно точно.

### Реф 8: Heavy Blog Is Heavy — редизайн 2022
**Источник:** https://heavyblogisheavy.com / Behance «metal magazine web design»

Металл-медиа с тёмным фоном, чёткая типографика без засечек, карточки с живыми фото групп, один акцентный цвет для кнопок и тегов — строго на весь сайт. Системность в деталях.

**ЧТО БРАТЬ:** Принцип единого акцентного цвета. На нашем сайте — один цвет для кнопок, счётчиков, FOMO-меток.
**ЧТО НЕ БРАТЬ:** Журнальную структуру (много колонок, много текста) — нам нужен лендинг, не медиа.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Металлические медиа умеют держать баланс темноты и читабельности. Это редкий навык — берём как технический реф.

# МУДБОРД 2: CONTENT / UX

### Реф 9: Primavera Sound — структура лайнапа
**Источник:** https://www.primaverasound.com (архив 2023-2024)

Лайнап-блок: фильтрация по дню и сцене, карточки артистов с фото, ховер — всплывает короткий тег жанра. Mobile — свайп по дням. Фильтрация без перезагрузки страницы.

**ЧТО БРАТЬ:** Карточный подход к лайнапу — фото + имя + сцена/время. Mobile-first навигация по блоку.
**ЧТО НЕ БРАТЬ:** Пастельные цвета Primavera — их айдентика не для нас.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** У нас 12 групп + хедлайнер. Показать список текстом — убить интерес. Карточки с фото = каждая группа становится реальной для незнакомого пользователя.

### Реф 10: Boiler Room — FOMO-механики
**Источник:** https://boilerroom.tv

Sticky-header с таймером до события. При скролле кнопка «КУПИТЬ БИЛЕТ» остаётся видна всегда. Рядом: «Осталось X мест» с обновлением. Кнопка никогда не уходит с экрана — конверсия в билет возможна в любой момент.

**ЧТО БРАТЬ:** Sticky CTA + счётчик оставшихся мест (пусть статичный, но присутствующий).
**ЧТО НЕ БРАТЬ:** Агрессивный попап «Вы уходите?» — для нашей аудитории это провал, металлисты ненавидят давление.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Продать 1000 билетов за 3.5 месяца без FOMO-механик — тяжело. Boiler Room показывает как делать это без дешёвых триггеров.

`[UX]`

### Реф 11: Glastonbury — мобильная навигация
**Источник:** https://www.glastonburyfestivals.co.uk (mobile view)

Mobile hamburger → full-screen overlay меню на тёмном фоне, пункты крупным шрифтом. Кнопка «БИЛЕТЫ» отдельным цветом прямо в меню — не надо закрывать меню чтобы купить.

**ЧТО БРАТЬ:** Full-screen overlay для mobile nav. Кнопка покупки выделена прямо в меню.
**ЧТО НЕ БРАТЬ:** Размер текста — у Glastonbury мелко, нам нужно крупнее для аудитории 35-45.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** 70%+ трафика на мероприятие приходит с мобайла. Мобильная навигация — не «адаптация», это основной интерфейс.

`[UX]`

### Реф 12: Bandcamp — тёмный виджет покупки
**Источник:** https://bandcamp.com — тёмный тип виджета / Behance «ticket widget dark»

Компактный виджет с тёмным фоном: цена, кнопка, минимальная информация. Вставляется в любой блок сайта без нарушения общего дизайна. Тёмный embedded-виджет не «выпрыгивает» белым прямоугольником.

**ЧТО БРАТЬ:** Принцип стилизации виджета TicketsCloud под тёмную тему сайта через кастомный CSS.
**ЧТО НЕ БРАТЬ:** Весь UX Bandcamp — он про музыку, не про ивенты.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** TicketsCloud даёт белый виджет по умолчанию. Если его не стилизовать — он сломает атмосферу тёмного сайта в ключевой момент покупки.

`[UX]`

### Реф 13: Riot Fest — информационная архитектура
**Источник:** https://www.riotfest.org

Один лендинг с якорями: Hero → Лайнап → Расписание → Билеты → FAQ → Карта. Все разделы доступны из sticky nav. Ни одного отдельного URL — всё на одной странице. SPA-структура — пользователь не теряется, не кликает на другие страницы.

**ЧТО БРАТЬ:** Одностраничный подход с якорной навигацией. Минимум внешних страниц.
**ЧТО НЕ БРАТЬ:** Пёстрый панк-дизайн Riot Fest — у нас другая эстетика.
**ПОЧЕМУ РАБОТАЕТ ДЛЯ НАС:** Для первого фестиваля без истории лендинг эффективнее многостраничного сайта. Удержание внимания важнее глубины структуры.

# АНТИ-МУДБОРД — ЧТО НЕ ДЕЛАТЬ

### Анти-реф 1: Корпоративный белый фон
**Источник:** Любой event.ru / kassir.ru — страница мероприятия

Белый или светло-серый фон, синие кнопки, стоковые улыбающиеся фото, «Добро пожаловать на наше мероприятие». Сайт подходит для корпоратива.

**ПОЧЕМУ НЕ ПОДХОДИТ:** Металлисты 35-45 лет закрывают такой сайт за 3 секунды. Ощущение «я не туда попал». Белый фон — не наш мир. Если дизайн подошёл бы для HR-портала — переделывать.

`[Anti]`

---

### Анти-реф 2: AI-арт как фон или иллюстрации
**Источник:** Midjourney / Stable Diffusion — запрос «metal festival poster»

Гиперреалистичный череп с огнём, «эпичная» сцена, которой никогда не было, идеально симметричный «металлический» воин. Выглядит «слишком идеально» или «слишком фэнтезийно».

**ПОЧЕМУ НЕ ПОДХОДИТ:** Наш дизайнер Давид категорически против AI-контента. Аудитория считывает AI-арт мгновенно — убийца доверия. «Не могли нормальных фото найти — сгенерировали». Нет исключений.

`[Anti]`

---

### Анти-реф 3: Stock-photo концерты
**Источник:** Shutterstock / Getty — запрос «rock concert crowd»

Безымянная толпа с поднятыми руками, подозрительно чистая сцена, идеальное освещение, никакого пота и грязи. В кадре нет конкретных лиц или конкретных групп.

**ПОЧЕМУ НЕ ПОДХОДИТ:** Наша аудитория была на сотнях концертов. Стоковое фото видно сразу. «Это не наши, это купленная картинка». Доверие -100. Если в фото нет конкретной группы или конкретного места — не наше.

`[Anti]`

---

### Анти-реф 4: Перегруз элементами
**Источник:** kassir.ru, afisha.ru — страницы событий

Баннер + несколько попапов + бегущая строка + 5 колонок с информацией + реклама + всплывающее окно при входе. На первом экране больше трёх смысловых блоков.

**ПОЧЕМУ НЕ ПОДХОДИТ:** Металлисты ценят прямоту. Перегруженный сайт = неуважение к времени. «Что тут вообще главное?» Если первый экран не отвечает на вопрос «когда, где, купить» — переделывать.

`[Anti]`

---

### Анти-реф 5: Вылизанный премиум-стиль
**Источник:** Behance — поиск «luxury event website», Swiss design для ивентов

Много воздуха, тонкие линии, тонкий шрифт, нейтральные цвета, «элегантность», золотые акценты. Если дизайн подошёл бы для ювелирного магазина или luxury-отеля.

**ПОЧЕМУ НЕ ПОДХОДИТ:** Металлисты придут в майках с логотипами трэш-групп. Luxury-эстетика — сигнал «не для нас». Дистанция вместо принятия. Чистота и «взрослость» = чужое.

`[Anti]`

# ЦВЕТОВАЯ ПАЛИТРА

| Цвет | HEX | Назначение |
|---|---|---|
| Абсолютный чёрный | `#0a0a0a` | Основной фон |
| Тёмно-серый (поверхности) | `#1a1a1a` | Карточки, блоки |
| Акцентный (огонь/металл) | `#c8460a` | CTA-кнопки, счётчики, FOMO-элементы |
| Белый текст | `#f0f0f0` | Основной текст (не чистый белый — мягче) |
| Приглушённый серый | `#6b6b6b` | Вторичный текст, метки, даты |
| Металлический бликовый | `#8a8a8a` | Разделители, иконки |
| Аварийный красный | `#8b1c1c` | Только для «ПРОДАЖИ ЗАКРЫТЫ» или «SOLD OUT» |

**Принцип:** один акцентный цвет. Всё остальное — оттенки чёрного и серого. Акцент используется строго: кнопка «Купить», счётчик мест, дата феста в hero.

# ТИПОГРАФИКА

**Заголовки H1-H2:** Bebas Neue или Impact — caps lock, без засечек, агрессивный tracking
**Заголовки H3-H4:** Inter Bold — нейтральная читаемость
**Основной текст:** Inter Regular 16px — удобочитаемость важнее стиля
**Акцентные метки:** IBM Plex Mono — для счётчиков, технических данных («Осталось 847 мест»)
