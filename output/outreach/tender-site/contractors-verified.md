## Методология

- Инструмент: WebFetch (прямая проверка сайта) + WebSearch (поиск альтернативных доменов)
- Критерии:
  - **Актив** = сайт открывается + есть контакт для связи
  - **Под вопросом** = сайт работает / контакт не найден / нет релевантных кейсов в открытом доступе
  - **Закрыт** = ENOTFOUND / TLS error / нерелевантный сайт
- Дата проверки: 2026-04-03
- Примечание: портфолио ивент-кейсов для большинства студий требует ручной проверки через Behance/личный звонок — автоматически парсить не удаётся
