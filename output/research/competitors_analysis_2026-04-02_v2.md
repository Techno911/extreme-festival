Ключевые слова в коммуникации: «все жанры», «впервые в одном месте», «ярмарка», «отдых», «зоны восстановления после слэма».

### Посещаемость

- 2025 (первый фест): ~700 человек (данные из контекста проекта; публичных официальных цифр нет)
- 2026: прогноз неизвестен, но расширили до 3 дней и 38 групп — явный рост амбиций

### Сильные стороны Скрежета

1. **Первенство** — провели уже в 2025, есть лояльная аудитория, «узнаваемость»
2. **Масштаб** — 38 групп, 3 дня: для части аудитории «больше = лучше»
3. **Охват жанров** — широкая сеть: кто-то придёт на death, кто-то на gothic
4. **Инфраструктура** — зоны отдыха, фудкорт, мерч-ярмарка (чётко отработанный формат)
5. **Сроки** — за 3 недели до нас, не в один день (можно посетить оба)
6. **Площадка** — Pravda Summer: открытая, летняя атмосфера

### Слабые стороны Скрежета

1. **«Котёл жанров»** — Thrash + Gothic + Pop-Metal вперемешку. Для треш-металлиста это оскорбление вкуса. Б.А.У. и Illidiance в одном лайнапе с Annodomini — не тот контекст.
2. **Самодеятельность** — 38 групп, значит многие — локальная клубная сцена без нацтура
3. **Open-air = зависимость от погоды** — июнь в Москве непредсказуем
4. **Фуд-корт и «зоны отдыха»** — ощущение ярмарки, не музыкального события
5. **Нет иностранного хедлайнера** — исходя из открытых данных, лайнап полностью российский
6. **Позиционирование «всё подряд»** — «полный спектр тяжёлой музыки» = нет идентичности

### Контент-стратегия (по открытым данным)

- Telegram-канал: t.me/skrezhet_public — публичный, контент доступен
- Активность: анонсы билетов, объявления групп (судя по публичным постам)
- Тон: официально-анонсовый, нет «своего голоса»
- ВКонтакте: прямой URL не найден (продаётся через pravdaconcert.ru — площадка)
- Аудитория ВК/ТГ: точные данные по подписчикам установить не удалось

## 2. КИТАЕВ (11 июля 2026)

### Статус исследования

**Конкретное событие 11 июля 2026 НЕ НАЙДЕНО ни на одной афишной платформе.**

Проверено: afisha.ru, afisha.yandex.ru, kudago.com, ticketland.ru, myrockshows.com, bandsintown.com, songkick.com, concertinfo.ru, allfest.ru — нет.

### Что известно о Grima

- Grima — российская группа, жанр **атмосферный блэк-метал** (Сибирь, братья Vilhelm/Morbius, с 2014)
- Booking: Madstream Booking Agency (madstreambooking.com)
- Последний найденный концерт в Москве: декабрь 2023 (клуб Urban)
- Tuров 2026 в открытых источниках не анонсировано

### Гипотеза

«Китаев» — предположительно **организатор/промоутер**, не название фестиваля. Возможно:
- Промоутерский бизнес Китаева под Grima (блэк-метал направление)
- Частное событие без публичного анонса (рано для апреля 2026?)

**Требует уточнения у Жени:** что конкретно за событие, есть ли анонс в ВК/ТГ Китаева, какая площадка «Grima» — это клуб или группа.

### Стратегическая оценка

- Жанр блэк/атмосферный метал **минимально пересекается** с нашей треш-аудиторией
- Прямой конкуренции за одну аудиторию нет
- НО: часть «общей металл-аудитории» выберет между двумя событиями 11 июля
- При нашем лайнапе (Master + Stillbirth) Эстрим Фест явно выигрывает у неанонсированного события

## 5. СТРАТЕГИЧЕСКИЕ ВЫВОДЫ ДЛЯ ЭСТРИМ ФЕСТА

### Наши преимущества

1. **Иностранный хедлайнер** — Stillbirth из Германии. Это само по себе событие. Скрежет не может этим похвастаться. Аргумент: «когда последний раз немцы приезжали?»

2. **Чистота жанра** — треш и кроссовер-треш vs «весь спектр». Для настоящего треш-металлиста Эстрим Фест — это «для своих», а Скрежет — «для всех», то есть ни для кого конкретно.

3. **Мы дешевле** — 2 900 ₽ стартовая vs 3 500 ₽. При равном или лучшем лайнапе это аргумент.

4. **Master** — легенда российского треша. Стержень для «своих в тусовке» и «паломников».

5. **Не в один день со Скрежетом** — за 3 недели разница. Теоретически можно посетить оба. Нет прямого конфликта.

### Окна возможностей

1. **Контент от групп** — Скрежет явно работает в анонсовом режиме. Мы можем выиграть на контент-маркетинге: репетиции, backstage, видеоприглашения от Master, Stillbirth.

2. **Нарратив «отбор vs котёл»** — не сравниваем публично, но в собственном контенте: «12 групп, каждая — по делу». Намёк понятен аудитории без слова «Скрежет».

3. **Иностранный хедлайнер как инфоповод** — анонс Stillbirth это PR-событие. Немецкий брутал-дэт в Москве. Аккредитуем metal-медиа именно на этот факт.

4. **Ценовой аргумент в ранних продажах** — старт 2 900 ₽ при анонсе — пока Скрежет продаёт от 3 500 ₽.

5. **Скейт-зона** — уникальный элемент которого нет у Скрежета. Отдельная аудитория.

### Риски

1. **Скрежет уже продаёт билеты** — раньше нас. Часть бюджета аудитории уже потрачена.
2. **38 групп — убедительный объём** — для неофита «больше = лучше».
3. **Лояльная аудитория с 2025** — у них уже есть «постоянники».
