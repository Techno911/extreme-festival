## 1. СТРАТЕГИЯ

**Цель:** 8-10 кружков/постов-рекомендаций к середине июня 2026.

Логика: не количество, а доверие. Один кружок от Леоса = 50 билетов.
Три кружка от незнакомых = ноль.

**Принцип сегментации:**
- Металлисты → убеждают металлистов
- Скейтеры → убеждают скейтеров
- Один для всех = ни для кого

**Приоритет:** ICE (Impact × Confidence / Effort). Начинаем с HIGH.

**Что мы даём:**

- Имя/ник в посте «наши амбассадоры» на странице фестиваля

## 5. FOLLOW-UP ЦЕПОЧКА

### Follow-up #1 (через 3 дня)

```
Привет [имя], писал про Эстрим Фест. Видел сообщение?
Если вопросы — отвечу. Если нет — просто дай знать.
```

### Follow-up #2 (через 7 дней)

```
[Имя], уточняю насчёт Эстрим Феста.

За это время: [конкретная новость — «объявили хедлайнера» /
«продали 100 билетов» / «5 амбассадоров уже подтвердили»].

Если формат кружка неудобен — можно просто фото + текст который я напишу.
Тебе остаётся согласовать и опубликовать.
```

### Follow-up #3 (через 14 дней — финальный)

```
[Имя], последний раз — не хочу надоедать.

Минимальный формат: репост нашего поста с коротким комментарием.
2 минуты + 2 билета.

Если нет — без обид. Спасибо что прочитал.
```

> **После 3-го follow-up без ответа — не пишем. Идём к следующему кандидату.**

## 6. ИНСТРУКЦИЯ ДЛЯ АМБАССАДОРА (после согласия)

### Кружок — 3 варианта текста (амбассадор выбирает)

**Вариант 1 — металлист:**
«Привет, я [имя / группа]. 11 июля в Москве — Эстрим Фест.
[Группа X] там будет. Серьёзный фест с серьёзным звуком.
Рекомендую. Ссылка на билеты — в описании.»

**Вариант 2 — скейтер:**
«Привет, это [имя]. 11 июля — Эстрим Фест, Москва.
Скейт-зона + живой металл. Атмосфера нормальная, я буду.
Ссылка в описании.»

**Вариант 3 — свободный:**
Ничего из выше. Свои слова. Главное: дата, город, где билеты.

**Технические требования:**
- 15-30 секунд
- Основная камера (не фронталка)
- Хороший свет, нормальный звук
- Без фильтров TikTok
- Без скрипта («ну вот, записываю видео для...» — не надо)

### Пост — готовые варианты

- «Рекомендую Эстрим Фест — 11 июля, Москва. [Группа X] там будет. [ссылка]»
- «11 июля. Москва. Эстрим Фест. Там будет [группа] — этого достаточно. [ссылка]»
- «Буду на Эстрим Фесте 11 июля. Треш-металл, скейт-зона, серьёзный лайнап. [ссылка]»
