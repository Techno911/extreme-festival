
## CONTINGENCY

| Проблема | Решение |
|---|---|
| Нет 15 КП на тендер | Снизить до 10. Расширить критерии. Добавить фрилансеров |
| Сайт задерживается | За 2 дня до дедлайна — ping. Если срыв — параллельно запасной из топ-3 |
| Нет видеографа в бюджете | Женя + друг + iPhone 15 Pro + ZoonKit (аренда ~5к/день) |
| Мало амбассадоров согласились | Снизить порог: любой человек с 1к+ подписчиков |
| Продажи <200 к июню | Перебросить весь бюджет на посевы. Розыгрыш «50 билетов со скидкой 20%» |
| Конкурент переманивает | Не воевать. Усиливать уникальность: «иностранец + Master» |
