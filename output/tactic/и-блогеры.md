## 2. ФОРМАТЫ КОЛЛАБОРАЦИИ

### Формат 1: Обзор лайнапа

**Кому:** YouTube-каналы (макро/средние)
**Что:** Блогер разбирает 12 групп — кто крут и почему

**Инфопакет от Writer (отправляем блогеру):**
По каждой из 12 групп:
- Название группы
- Жанр + поджанр
- 2-3 предложения «чем уникальны / почему стоит услышать»
- Ссылка на лучший клип/запись
- Что ожидать от живого выступления

**Шаблон письма блогеру:**
```
Привет [имя]!

Ты делаешь контент по металу — значит интересно. 11 июля в Москве
Эстрим Фест: 12 групп (треш/кроссовер) + иностранный хедлайнер + Master.

Хотим предложить сделать обзор лайнапа для твоей аудитории.
Подготовим инфопакет по каждой группе — тебе останется снять видео.
Взамен: 2 билета + упоминание в наших постах.

Интересно? Пришлю детали.
```

---

### Формат 2: Розыгрыш билетов

**Кому:** Все уровни, все платформы
**Что:** «Подписка + репост = шанс выиграть 2 билета на Эстрим Фест»

**Текст розыгрыша (готовые варианты):**

Вариант А (для метал-каналов):
```
Разыгрываем 2 билета на Эстрим Фест — 11 июля Москва.
12 серьёзных групп. Иностранный хедлайнер. Master.

Условия: подписаться на [канал] + репостнуть этот пост.
Результаты: [дата], рандом.
```

Вариант Б (для городских каналов):
```
Москва, 11 июля. Металл-фестиваль. 2 сцены, скейт-зона, 1400 мест.

Разыгрываем 2 билета.
Условия: подписка + репост.
```

---

### Формат 3: Реакция на трейлер

**Кому:** YouTube-каналы (когда трейлер готов — июнь)
**Что:** Отправляем трейлер до публикации, просим снять реакцию

**Письмо:**
```
[Имя], трейлер Эстрим Феста выходит [дата]. Хочу показать тебе заранее.

Было бы круто если бы ты снял реакцию — 5-10 минут, твои мысли.
Если зайдёт — публикуем одновременно (и у тебя, и у нас).

Взамен: 2 билета + упоминание. Интересно?
```

---

### Формат 4: Нативный пост (для ТГ-каналов/пабликов)

**Что:** Не «реклама», а «свой человек рекомендует»

**Варианты текстов (Writer адаптирует под тон каждого канала):**

Для металл-канала:
```
11 июля в Москве. Эстрим Фест. Видел лайнап — там [X групп].
Master + иностранный хедлайнер + ещё 10. Серьёзно.
Билеты пока есть: [ссылка с UTM]
```

Для городского канала:
```
На что идти 11 июля в Москве: Эстрим Фест.
Металл-фестиваль. 2 сцены, скейт-зона, живой звук.
Необычно. Вот: [ссылка с UTM]
```

---

### Формат 5: Приглашение на фест

**Кому:** Блогеры с камерой (YouTube, ТГ-видео)
**Что:** Аккредитация на фест для съёмки влога

**Условия аккредитации:**
- 2 билета (для блогера + оператора/друга)
- Разрешённые зоны съёмки (обговаривается заранее)
- Ограничения: нельзя публиковать съёмку с 10:00 до 12:00 [x-дата] без согласования
- Wi-Fi для блогера на месте
- Контакт на площадке: Женя или волонтёр

## 6. UTM-ССЫЛКИ (обязательно!)

Каждому блогеру — уникальная ссылка:
```
extremefest.org?utm_source=[блогер_имя]&utm_medium=[формат]&utm_campaign=bloggers
```

Примеры:
- Леос: `?utm_source=leos_hellscream&utm_medium=vk_post`
- Telegram-канал X: `?utm_source=tg_metalchannel&utm_medium=telegram`
- Розыгрыш: `?utm_source=contest_vk&utm_medium=repost`

**Зачем:** Без UTM — нельзя понять кто реально привёл билеты. Это деньги на ветер.
