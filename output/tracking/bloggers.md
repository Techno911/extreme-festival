## ИНСТРУКЦИЯ ПО UTM-ССЫЛКАМ

Для каждого блогера создаётся уникальная ссылка:
```
https://tickets.extremefest.ru/?utm_source=[платформа]&utm_medium=influencer&utm_campaign=[имя_блогера]
```

**Шаблоны:**
- ВКонтакте: `utm_source=vk&utm_medium=influencer`
- Instagram: `utm_source=instagram&utm_medium=influencer`
- Telegram: `utm_source=telegram&utm_medium=seeding`
- YouTube: `utm_source=youtube&utm_medium=influencer`

Отслеживание: TicketsCloud показывает источник покупки. Сводить раз в неделю.
