# 8 шаблонов коммуникации с партнёрами

## 1. Предложение короткое (для всех)

```
Тема: Инфопартнёрство — Эстрим Фест 11 июля, Москва

Привет!

Организую Эстрим Фест — металл-фестиваль 11 июля в Москве.
12 российских групп + иностранный хедлайнер, 1400 мест.

Предлагаю инфопартнёрство — бартер, без денег:
[уровень партнёрства — базовый/стандарт/премиум]

Короткое описание уровней в медиаките: [ссылка]

Если интересно — напишите, пришлю подробности.

Женя Аржевский, организатор
[TG/email]
```

## 2. Предложение расширенное (для радио и брендов)

```
Тема: Инфопартнёрство — Эстрим Фест × [название]

Привет!

Меня зовут Женя, я организую Эстрим Фест — металл-фестиваль в Москве 11 июля 2026 года.

О фестивале:
— 12 групп, отобранных по качеству (не по популярности)
— 1 иностранный хедлайнер
— Скейт-зона на улице
— 1400 мест, целевая аудитория 18-45 лет

Почему предлагаю именно вам: [конкретная причина — аудитория, тематика, опыт прошлых коллабов]

Что предлагаем (Премиум партнёрство):
— Логотип на сайте, афише, баннере площадки
— 6 билетов + зона на площадке
— Совместный контент-проект (розыгрыш / серия постов)

Что просим:
— [конкретно что нужно от этого партнёра]

Медиакит: [ссылка]

Можем пообщаться 15 минут чтобы обсудить детали?

Женя
[TG/email]
```

## 3. Скрипт телефонного звонка (радиостанции, магазины)

```
Приветствие:
«Добрый день, это Женя Аржевский, организатор фестиваля Эстрим Фест.
Я хотел бы поговорить с [нужным человеком] по поводу возможного партнёрства.
Это займёт буквально 3 минуты.»

Суть:
«Мы проводим металл-фестиваль 11 июля в Москве. 12 групп, иностранный хедлайнер, 1400 мест.
Ищем инфопартнёров — рассматриваем бартер на билеты.»

Вопрос:
«Вам было бы интересно рассмотреть такое партнёрство? Могу прислать медиакит.»

[При интересе: уточнить контакт → выслать медиакит + предложение]

Ответы на возражения: см. objections.md
```

```
Медиакит: [ссылка]

Как тебе?
```

## 6. Follow-up (через 5 дней)

```
Привет! Писал про Эстрим Фест [N] дней назад. Видели?

Обновление: [новость — подтвердили хедлайнера / ещё X партнёров / продали X билетов].

Предложение в силе. Медиакит: [ссылка]
```

## 7. Ответ на «сколько стоит?»

```
Партнёрство бесплатное — бартер на билеты.

Если нужна именно оплата — готов обсудить. Какой формат и условия?
У нас есть небольшой бюджет на платные размещения.
```
