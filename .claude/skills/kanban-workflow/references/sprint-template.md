# Шаблон спринт-планирования — Эстрим Фест

> Спринт = 1 неделя (пн–пт). Планирование: каждый понедельник до 12:00.
> Заполняет CMO или Head-агент при получении задачи от Жени.

---

## Недельный ритм

| День | Действие |
|------|---------|
| **Пн** | Планирование: взять задачи из бэклога, проверить WIP, уточнить блокеры |
| **Вт–Чт** | Работа. Если задача выполнена досрочно — брать следующую из бэклога |
| **Пт до 17:00** | Все задачи → «На проверке» или «Не принято». Бэклог следующего спринта |
| **Пт 17:00** | Дедлайн. После — только экстренное |

- [ ] Ретроспектива заполнена
