# Мерч

## Принципы
- Мерч = дополнительный доход + живая реклама на теле покупателя
- Дизайн — только Оля/Давид. Никакого AI-арта
- Продажа: на площадке + онлайн (если позволяет производство)

## Ассортиментная матрица
@references/assortment.md

Базовый набор:
| Позиция | Тираж | Цена |
|---|---|---|
| Футболка (унисекс, S-XXL) | 100 шт | 1500-2000₽ |
| Худи | 50 шт | 3000-3500₽ |
| Нашивка | 200 шт | 200-300₽ |
| Значок | 300 шт | 100-150₽ |
| Плакат А2 | 100 шт | 300-500₽ |

## Стратегия производства
- Топ качество важнее цены — это имиджевый мерч
- Производство под предзаказ — минимизировать остатки
- Ernie Ball уже партнёр — можно просить мерч или скидку

## Дизайн-бриф
@references/design-brief.md — ТЗ для Давида/Оли

## Gotchas
- Не переусердствуй с тиражом — остатки = замороженные деньги
- Ограниченная серия + «только на площадке» = коллекционная ценность
