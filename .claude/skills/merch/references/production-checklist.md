# Чеклист производства и продаж мерча

> Общий бюджет производства: ~100 000 ₽ (из раздела В)
> Дедлайн заказа: **1 июня** (чтобы получить к 25 июня, резерв 2 недели)
> Ответственный: Женя → макеты от Давида/Оли

---

## ЭТАП 1: ДИЗАЙН (дедлайн — 1 мая)

- [ ] Давид получил ТЗ по дизайн-бриф (`references/design-brief.md`)
- [ ] Логотип финализирован (векторный файл .ai / .svg)
- [ ] Маскот утверждён (вариант для значков: медведь-зомби / лого / лайнап)
- [ ] Макеты для каждого SKU готовы:
  - [ ] Футболка — шелкография 2 цвета (лицевая сторона)
  - [ ] Худи — шелкография 2 цвета
  - [ ] Бандана — набивка (лого + маскот)
  - [ ] Значки — 3 вида (d=58мм), каждый отдельным файлом
- [ ] Все макеты в форматах: CMYK, 300 dpi, без шрифтов (кривые)
- [ ] Раскладка по размерам футболок согласована (S:10 / M:30 / L:35 / XL:20 / XXL:5)

## ЭТАП 2: ВЫБОР ПОДРЯДЧИКА (дедлайн — 15 мая)

### Что сравнивать при выборе

| Критерий | Обязательно | Желательно |
|----------|------------|-----------|
| Срок производства | ≤21 день с момента оплаты | ≤14 дней |
| Предоплата | ≤50% | 30% |
| Шелкография | Минимум 2 цвета включены | 3+ цвета без доп. платы |
| Портфолио с металом / рок-мерчем | Есть примеры | Работали с музыкантами |
| Пробный тираж | Есть возможность (5-10 шт) | Бесплатный образец |
| Доставка | Включена в цену или до 2 000 ₽ | Курьер до адреса |
| Контроль качества | Фото готового до отгрузки | Видеозвонок с примеркой |

### Где искать подрядчика

1. **ВКонтакте:** поиск «печать на футболках Москва» → группы с реальными отзывами
2. **Авито:** частники дешевле, но риск задержки — только с предоплатой ≤30%
3. **Kwork.ru:** монтаж + печать пакетом
4. **Рекомендация из тусовки:** спросить Женю Косырева — у метал-групп есть проверенные

## ЭТАП 3: ЗАКАЗ (дедлайн — 1 июня)

- [ ] Подрядчик выбран, договор / накладная подписаны
- [ ] Предоплата (≤50%) переведена
- [ ] Макеты переданы подрядчику в финальном виде
- [ ] Дата готовности зафиксирована письменно: **не позднее 25 июня**
- [ ] Договорились о фото готового мерча до отгрузки

## ЭТАП 4: ПРЕСЕЙЛ (июнь)

**Цель:** продать 20-30% тиража до феста → кэш + данные по размерам

### Механика пресейла

1. **Пост в ВК** с фото готового мерча (не скетча — реального изделия)
2. CTA: «Заказ в DM: напиши размер + кол-во. Самовывоз на феству или доставка +200₽»
3. **Ограничение:** только для купивших билет (верификация через TicketsCloud ID)
4. Предоплата при пресейле: полная стоимость
5. Срок пресейла: **10–25 июня**

### Чеклист пресейла

- [ ] Фото финального мерча готово (от Давида или своей съёмкой)
- [ ] Пост опубликован с реальными фото
- [ ] DM-скрипт готов (что отвечать на заявки)
- [ ] Таблица заказов создана (имя / размер / кол-во / оплачено)
- [ ] Оплата через ЮКассу или Сбер перевод (не наличными заранее)

## ЭТАП 5: ПРОДАЖИ НА ФЕСТЕ (11 июля)

### Организация точки продаж

- [ ] Стол / стойка в фойе или у входа — согласовать место с клубом заранее
- [ ] Продавец выбран (из тусовки, знает мерч): **5-8к₽ + %**
- [ ] Наличные + терминал оплаты (SberPay / Tinkoff)
- [ ] Ценники готовы (крупный шрифт, видно издалека)
- [ ] Упаковка (пакеты с лого или крафт-пакеты)
- [ ] Инвентаризационный список: что взяли = что привезли обратно

### Раскладка цен (онсайт = выше пресейла)

| SKU | Пресейл | Онсайт |
|-----|---------|--------|
| Футболка | 1 300₽ | 1 400₽ |
| Худи | 2 000₽ | 2 200₽ |
| Бандана | 300₽ | 350₽ |
| Значок | 150₽ | 200₽ |

### Выкладка

- Футболки/худи: на вешалках по размерам, размерные бирки видны
- Значки: на планшете или в коробке-«витрине»
- Бандана: у кассы рядом с зоной оплаты (импульсная покупка)
- «Комплект» (футболка + бандана + значок): со скидкой 150₽ — озвучивать вербально

## ФИНАНСОВАЯ СВОДКА

| Сценарий | % продаж | Выручка | Вложения | Прибыль |
|----------|----------|---------|---------|---------|
| Пессимизм | 40% | ~128 000₽ | 99 250₽ | ~29 000₽ |
| Базовый | 60% | ~192 300₽ | 99 250₽ | ~93 000₽ |
| Оптимизм | 80% | ~256 000₽ | 99 250₽ | ~157 000₽ |

**Правило:** мерч не должен уходить в минус даже при 40% продаж. ✅ (по текущей матрице)

---

## КРАСНЫЕ ФЛАГИ

- Макет от Давида не готов к 10 мая → сдвигается весь план → **не экономь на сроках**
- Предоплата 100% незнакомому подрядчику → **не давать**
- Тираж слишком большой (>200 футболок) → **нет, 100 — это максимум для первого раза**
- Продажа через сторонних перекупщиков → **запрет** (роняют цену, как с билетами)
