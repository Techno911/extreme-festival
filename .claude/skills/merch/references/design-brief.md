# Дизайн-бриф для мерча (ТЗ для Давида/Оли)

> Этот документ — задание для художника, не для AI.
> Никакой AI-генерации. Всё рисуется руками.

## Концепция

Мерч фестиваля = не просто сувенир, а вещь которую хочется носить.
Ориентир: стиль 80-х метал + современный streetwear.
Аудитория: 18-45 лет, металлисты + скейтеры.

## Требования к дизайну

### Что должно быть на каждой позиции
- Название «Эстрим Фест» (или «Extreme Fest»)
- Дата: 11 июля 2026
- Опционально: Москва

### Стиль
- Грубый, живой — не лощёный
- Типографика как в оформлении 80-х метал-альбомов
- Можно иллюстрации от Оли (скетч-стиль)
- НЕ: глянцевый, «корпоративный», нейтральный
- НЕ: AI-сгенерированные изображения

### Цвета
- Чёрный + 1-2 акцентных цвета (обсудить с Давидом)
- Для скейтерской серии: другая палитра (уличный стиль)

## ТЗ по позициям

### Футболка
- Печать: шелкография (не сублимация — выгорает)
- Фасон: унисекс, оверсайз (не приталенная)
- Расположение: принт на груди + маленькое лого сзади
- Количество цветов в печати: до 3

### Нашивка
- Формат: прямоугольная или фигурная, с вышивкой
- Сюжет: лого фестиваля / символ (обсудить с Давидом)
- Способ крепления: термоприклеивание + прошивка

### Значок
- 44мм диаметр
- Разные варианты дизайна (3-4 штуки)

### Плакат
- А2, офсетная печать
- Лайнап + дата + место + визуал
