# Маппинг колонок — Google Sheets «Контентная»

> Источник правды: `notifier/sheets.js`, константа `COL` (строка ~32).
> Строки 1–2: заголовки блоков. Строка 3: названия колонок. Строки 4+: данные.

## Полная таблица (A–Q)

| Колонка | Константа | Название в таблице | Тип | Кто заполняет | Примечания |
|---------|-----------|-------------------|-----|--------------|------------|
| **A** | `DAY` | День | Текст | Авто | Пн / Вт / Ср / Чт / Пт / Сб / Вс |
| **B** | `DATE` | Дата | Дата | Авто из файла | Формат DD.MM.YYYY в таблице |
| **C** | `RUBRIC` | Рубрика | Текст | Авто | Из мета-блока черновика |
| **D** | `SID` | ID | Текст | Авто | S-01, S-02, … S-23 |
| **E** | `STAGE` | Этап воронки | Текст | Авто | Привлечение / Прогрев / Продажа / Лояльность |
| **F** | `BEACON` | Маяк | Текст | Женя вручную | Стратегический акцент — не трогать агентам |
| **G** | `MEDIA_LINK` | Ссылка на медиа | URL | Женя/публикатор | Ссылка на фото/видео в облаке |
| **H** | `TEXT` | Текст | Длинный текст | Агент | Полный текст поста |
| **I** | `TEXT_GOAL` | Цель текста | Текст | Агент | Продающая часть А (ситуация/боль) |
| **J** | `SELLING` | Продающая часть А→Б | Текст | Агент | «было X → стало Y» |
| **K** | `MECHANIC` | Как сделать интересным | Текст | Агент | Механика вовлечения |
| **L** | `FORMAT` | Формат | Текст | Агент | Пост ВК / Пост ТГ / Reels / Stories / Карусель |
| **M** | `REACH_PLAN` | Охват план | Число | Агент | Ожидаемый органический охват |
| **N** | `REACH_FACT` | Охват факт | Число | Женя после публикации | — |
| **O** | `ERR_PLAN` | ERR% план | Процент | Агент | Engagement Rate, % |
| **P** | `ERR_FACT` | ERR% факт | Процент | Женя после публикации | — |
| **Q** | `STATUS` | Статус | Dropdown | Агент / Женя | Бэклог → Черновик → На проверке → Одобрен → Опубликован |

## Колонки R и дальше — ЗАПРЕТ НА ЗАПИСЬ

| Диапазон | Содержание | Почему нельзя трогать |
|----------|-----------|----------------------|
| R–? | Баланс воронки (авто) | Формулы считают распределение Привлечение/Прогрев/Продажа/Лояльность |
| T–Y | Справочники | Допустимые значения для дропдаунов |
| AB+ | Дополнительные расчёты | Формулы сводной статистики |

## Пример строки данных (строка 4+)

```
A: Пн
B: 06.04.2026
C: Группа недели
D: S-01
E: Привлечение
F: (пусто — заполнит Женя)
G: (пусто — при публикации)
H: [полный текст поста]
I: Не знаешь кто выступит — узнай сейчас
J: Не знаешь группу → откроешь для себя новый метал
K: Верю/не верю
L: Пост ВК
M: 3000
N: (пусто)
O: 4
P: (пусто)
Q: Черновик
```

## Код для прямой записи через Sheets API (если нужно без скрипта)

```javascript
const { sheets } = require('./notifier/sheets'); // экспортирует appendRows()

await sheets.appendRows('Контентная', [[
  day,      // A
  date,     // B — DD.MM.YYYY
  rubric,   // C
  sid,      // D
  stage,    // E
  '',       // F — маяк, оставить пустым
  '',       // G — медиа, оставить пустым
  text,     // H
  textGoal, // I
  selling,  // J
  mechanic, // K
  format,   // L
  reachPlan,// M
  '',       // N — факт, пусто
  errPlan,  // O
  '',       // P — факт, пусто
  'Черновик'// Q
]], 'USER_ENTERED');
```
