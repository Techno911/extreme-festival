# Стандарт тендера ЧиП — выжимка

## Принципы

1. **Тендер = бесплатный** — подрядчик участвует без предоплаты
2. **Параллельность** — минимум 10 КП одновременно, лучше 20-30
3. **Структурированность** — все КП по одному шаблону, иначе не сравнить
4. **Скоринг объективен** — критерии установлены до получения КП
5. **Одно КП = одно предложение** — не просим переделывать под другой формат

## 9 элементов тендерного пакета

1. **Маркетинговый бриф** — 5 блоков (контекст, задача, аудитория, KPI, ограничения)
2. **База подрядчиков** — 30+ с контактами, разбитых на категории
3. **Письмо первого контакта** — короткое, конкретное, с брифом во вложении
4. **Таблица скоринга** — заполняется после получения КП
5. **Критерии отсева** — быстрый фильтр до детального анализа
6. **Формула автоскоринга цены** — объективно сравнивает цены
7. **Шаблон обоснования выбора** — 1 страница А4 для итогового решения
8. **Чеклист онбординга** — договор, предоплата, Kanban, контакты
9. **Kanban-структура** — контроль этапов работ

## Письмо первого контакта (шаблон)

```
Тема: [Сайт/Трейлер/Мерч] для металл-фестиваля Эстрим Фест, 11 июля 2026

Привет [имя]!

Я Женя, организую металл-фестиваль «Эстрим Фест» (11 июля 2026, Москва).
Нам нужен [тип работы]. Во вложении — подробный бриф.

Если интересно — пришли КП до [дата]. Форма: [что должно быть в КП].

Женя
[контакт]
```

## Chеклист онбординга подрядчика

- [ ] Договор подписан (или хотя бы оферта)
- [ ] Предоплата 30% переведена (только после договора)
- [ ] Kanban создан: Задачи → В работе → На проверке → Готово
- [ ] Согласован формат еженедельного статуса
- [ ] Финальные исходники — наши (прописано в договоре)
- [ ] Дедлайны зафиксированы с штрафами/условиями

## Шаблон обоснования выбора (1 стр А4)

```
Подрядчик: [имя/компания]
Тип работ: [сайт/трейлер/мерч]

ПОЧЕМУ ВЫБРАЛИ:
1. Портфолио: [конкретная работа которая понравилась]
2. Понимание задачи: [что написали в КП что показало понимание]
3. Цена: [X₽, в рамках бюджета / ниже на Y%]
4. Скорость: [дедлайн X, нас устраивает]

РИСКИ:
[1-2 риска и как будем управлять]

СЛЕДУЮЩИЙ ШАГ:
[договор / предоплата / кик-офф]
```
