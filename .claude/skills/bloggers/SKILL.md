## Бюджет
Итого: 15-95к ₽ + 21-41 билет. Общий лимит бартера: не более 80 билетов.

## Ключевые KPI
- Охват посевов: 500к+ уникальных просмотров
- Переходы на билеты: 1000+ кликов (UTM-метки!)
- Публикаций: 30+

## Gotchas
- Каждый нативный текст → critic (не AI-звук) перед отправкой
- Для Леоса (Hell Scream, 500к) — отдельный бриф: не критиковать Master
- UTM-метка для каждого блогера — без этого непонятно кто реально привёл продажи
