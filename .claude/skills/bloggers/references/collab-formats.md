# 6 форматов коллабораций с блогерами

## 1. Обзор лайнапа

**Для кого:** YouTube макро/средние
**Что делает блогер:** разбирает 12 групп, рассказывает кто крут и почему

**Инфопакет от команды:**
- По каждой из 12 групп: 3 предложения (кто, жанр, чем крут)
- Ссылка на лучший клип
- Что ожидать на живом выступлении

**Почему работает:** глубокое погружение в лайнап → конверсия зрителей в покупателей

---

## 2. Розыгрыш билетов

**Для кого:** все платформы
**Механика:** «Подписка + репост = шанс выиграть 2 билета»

**Текст розыгрыша (шаблон):**
```
Разыгрываем [N] билетов на Эстрим Фест — 11 июля, Москва.

Условия:
1. Подпишитесь на [канал/группу]
2. Сделайте репост этой записи

Победители — [дата]. Случайный выбор.

Подробности о фесте: [ссылка на медиакит]
```

**Важно:** UTM-ссылка для каждого розыгрыша. Победителям доставить билеты красиво (конверт, не просто PDF).

---

## 3. Реакция на трейлер

**Для кого:** YouTube
**Механика:** отправить трейлер блогеру заранее → он снимает реакцию на камеру

**Подводка от команды:**
```
«Мы прислали трейлер Эстрим Феста [блогер] — вот его реакция.
[ссылка на ролик]»
```

**Требования к трейлеру:** минимум 2 минуты контента для реакции

---

## 4. Приглашение на фестиваль

**Для кого:** YouTube/ТГ (средние и микро)
**Что:** блогер приходит на фест, снимает влог

**Что обеспечивает команда:**
- Аккредитация (бейдж)
- Зоны для съёмки (без ограничений, кроме backstage)
- Контакт на площадке
- Wi-Fi, розетки

---

## 5. Нативный пост

**Для кого:** ТГ/ВК (средние и микро)
**Тон:** «свой человек рекомендует», НЕ рекламный

**Примеры нативных текстов:**

*Для метал-канала:*
```
Читаю про Эстрим Фест который будет 11 июля в Москве.
Лайнап неплохой — [группа X], [группа Y]. Без самодеятельности, нормальный отбор.
Цена вроде адекватная. Кто собирается?
```

*Для скейт-канала:*
```
Если кто ищет куда пойти 11 июля — на Эстрим Фесте будет скейт-зона.
Плюс метал, плюс атмосфера. Не типичный клубный вечер.
```

**Важно:** Writer готовит 3-5 вариантов под каждый канал → critic проверяет на AI-звучание.

---

## 6. Совместный контент (интервью)

**Для кого:** YouTube/подкасты
**Форматы:** интервью с организатором, с музыкантом из лайнапа

**Темы для интервью:**
- «Как собрать металл-фестиваль без самодеятельности» (с Женей)
- «Почему эта группа играет на Эстрим Фесте и не на Скреже» (с группой)
- «Как притащить иностранную группу в Россию в 2026» (если актуально)

**Ключевые сообщения (что нельзя не сказать):**
- Дата и место
- Почему лайнап отличается от конкурентов
- Иностранный хедлайнер (создаёт интригу)
- Где купить билеты
