# Пайплайн амбассадоров

## Цель
15+ кругляшков/фото к середине июня. Ноль рублей бюджета.

## База поиска (50+ контактов)

| Категория | Минимум |
|---|---|
| Российские метал-музыканты (не из лайнапа) | 15 |
| Скейтеры РФ | 10 |
| Мировые метал-звёзды | 10 |
| Мировые скейтеры | 5 |
| Смежные ниши (тату, стритвир, экстрим-спорт) | 10 |

## ICE-приоритизация
- Impact: сколько нашей ЦА увидит
- Confidence: есть ли связь через Женю Косырева?
- Ease: «написать в DM» vs «месяц с PR-агентством»

Пишем в порядке убывания ICE-балла.

## Пакет документов
- 3 pitch-письма RU (для звёзд / среднего / начинающих)
- 3 pitch-письма EN (для иностранцев)
- Follow-up цепочка: 3 дня → 7 дней → 14 дней
- Инструкция кругляшка: @references/video-brief.md
- Альтернатива: фото + текст

## Трекинг статусов
🔴 Не написали → 🟡 Написали → 🟠 Ответили → 🟢 Согласились → ✅ Материал получен → 📋 Согласован → 🎯 Опубликован

## Gotchas
- Iron Maiden / Tony Hawk — шанс <1%, но стоимость попытки = 0₽
- Фокус: российские музыканты, у которых есть связь через Женю Косырева
- Master — только уважительный тон
