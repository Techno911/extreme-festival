# Follow-up цепочка для амбассадоров

## Структура

```
День 0: Отправлен pitch
Day +3: Follow-up #1 (если нет ответа)
Day +7: Follow-up #2 (если нет ответа)
Day +14: Follow-up #3 финальный (если нет ответа)
```

---

## Follow-up #1 (через 3 дня)

**RU:**
```
[Имя], привет! Писал тебе про Эстрим Фест — получил?
Если не дошло — вот коротко: [1 строка суть].
Медиакит: [ссылка]
```

**EN:**
```
Hey [Name], just checking if my message about Extreme Fest reached you.
Quick recap: [1 line]. Media kit: [link]
```

---

## Follow-up #2 (через 7 дней)

**RU:**
```
[Имя], хотел уточнить — интересно ли вообще?

Кстати, обновление: [новый аргумент — например:
«Подтвердили хедлайнера», «уже 8 амбассадоров согласились»,
«продали 200 билетов»].

Форматы очень гибкие: кругляшок, фото + текст, просто репост.
```

**EN:**
```
Hey [Name], wanted to follow up one more time.

Quick update: [new argument — confirmed headliner / 8 ambassadors already on board].

Very flexible on format: video, photo post, or even just a repost.
```

---

## Follow-up #3 (через 14 дней — финальный)

**RU:**
```
[Имя], последнее сообщение по поводу Эстрим Феста.

Понимаю что много запросов. Если не актуально — окей, без обид.

Если формат кругляшка неудобен — можно просто фото с любым текстом.
Или просто репост анонса без слов. Любое участие ценно.

Если нет — всё равно спасибо, надеюсь увидеть тебя на фесте!
```

**EN:**
```
[Name], final message about Extreme Fest — no pressure at all.

If a video isn't your thing: photo + text works great, or even just a repost.
Any involvement is appreciated.

Either way, hope to see you there July 11!
```

---

## Правила

- Не более 3 follow-up (потом тишина)
- Между follow-up реальные новости/обновления — не просто «ты видел?»
- Follow-up через TG если изначально писали в TG
- Переключись на другой канал (email → Instagram DM) только если точно знаешь что там активнее
