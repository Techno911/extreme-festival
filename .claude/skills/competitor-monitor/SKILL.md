# Competitor Monitor

## Что мониторить еженедельно

| # | Параметр | Источник | Частота |
|---|----------|----------|---------|
| 1 | Анонсы лайнапа (новые группы, хедлайнер) | ВК, ТГ конкурентов | Еженедельно |
| 2 | Цены билетов (мин/макс, акции, скидки) | TicketsCloud, Kassir, MTC Live | Еженедельно |
| 3 | Подписчики ВК (абсолютное число + дельта за неделю) | vk.com/screzhmet | Еженедельно |
| 4 | Посты за неделю (кол-во + темы) | ВК, ТГ | Еженедельно |
| 5 | Engagement rate (лайки+репосты+комменты / подписчики) | ВК статистика | Еженедельно |
| 6 | Рекламные посевы (упоминания в других каналах) | ТГ-поиск, ВК | Еженедельно |
| 7 | Инфопартнёры и коллаборации | их посты + посты партнёров | По факту |
| 8 | Продажи билетов (если видно через ticketscloud/kassir) | TicketsCloud, MTC Live | Еженедельно |

- **ВК:** vk.com/screzhmet — основная страница, все посты и анонсы
- **TicketsCloud:** ticketscloud.com → поиск «Скрежет металла» или «скрежет»
- **Kassir.ru:** kassir.ru → поиск по названию
- **MTC Live:** mtslive.ru — если продают через них

### Китаев (конкурент 11 июля, площадка Grima)

- **Grima ВК:** vk.com/gruzia_grima (площадка) — смотреть их анонсы
- **TicketsCloud / MTC Live:** поиск по дате 11.07.2026 в Москве

## Формат таблицы сравнения

Используй шаблон из @references/comparison-template.md

Обязательные колонки:
- **Параметр** — что сравниваем
- **Эстрим Фест** — наши данные
- **Скрежет металла** — их данные
- **Источник / Дата** — откуда инфа и когда проверено
- **Наша позиция** — лучше / хуже / паритет + вывод

## Триггеры для алерта

Немедленно сообщи организатору (в Telegram через stdout) если:

| Триггер | Порог | Действие |
|---------|-------|----------|
| Конкурент объявил хедлайнера | Любой иностранный акт ИЛИ известный российский | АЛЕРТ: проверить не пересекается ли с нашим лайнапом |
| Конкурент снизил цены | Снижение > 15% или появился промокод | АЛЕРТ: пересмотреть нашу ценовую стратегию |
| Конкурент запустил акцию | Любой «early bird», «2 по цене 1», розыгрыш | АЛЕРТ: рассмотреть симметричный ответ |
| Резкий рост подписчиков | +500 за неделю (>10% текущей базы) | АЛЕРТ: найти причину — вирусный пост, посев, PR |
| Крупный инфопартнёр у конкурента | Радиостанция, СМИ с 50k+ охватом | АЛЕРТ: проверить не занята ли наша приоритетная площадка |
| Продажи конкурента >700 билетов | Если видно через открытые данные | АЛЕРТ: усилить собственные продажи |

## Алгоритм мониторинга (еженедельно)

1. Открыть vk.com/screzhmet — посмотреть посты за последние 7 дней
2. Проверить TicketsCloud / Kassir — изменились ли цены и количество доступных билетов
3. Посмотреть ТГ через tgstat.ru — упоминания «Скрежет», «Эстрим»
4. Проверить Grima ВК — новые анонсы на июль
5. Заполнить таблицу сравнения (@references/comparison-template.md)
6. Проверить триггеры алертов — есть ли что-то экстренное
7. Сохранить отчёт в `output/research/competitor-YYYY-MM-DD.md`
8. Резюме вывести в stdout (3-5 строк: что изменилось, есть ли алерты)

## Gotchas

- Китаев = артист/промоутер, НЕ название площадки. Grima — площадка.
- «Скрежет металла» в 2026 — open-air на Pravda Summer (Варшавское шоссе), НЕ в том же клубе. Уточняй каждый раз.
- Не считай подписчиков ВК как охват — у Скрежета могут быть накрутки. Смотри engagement.
- TicketsCloud показывает «осталось X мест» только если организатор включил отображение.
