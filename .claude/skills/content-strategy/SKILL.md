# SMM-концепция 3.6 (ЧиП) — ContentOps

## Цель

Написать конкретный пост по рубрикатору: текст, продающая часть, механика, формат. **Без этого skill ContentOps пишет «от балды».**

Чеклист перед публикацией: @references/post-checklist.md

---

## Обязательная структура каждого поста

**Каждый пост = продающий.** Даже «за кулисами» содержит CTA.

### 1. ХУК (первые 3 строки)
Решает: листать дальше или нет. Аудитория принимает решение за **2 секунды**.

Варианты хука:
- Вопрос: «Сколько групп ты уже пропустил только потому что не знал о них?»
- Шок-факт: «38 групп за 3 дня — и половина самодеятельность. Мы выбрали 12 из лучших»
- Цитата: «Металл — это не агрессия. Это точность. © [музыкант]»
- Противоречие: «Мы не хотим 1400 человек. Мы хотим 1000 правильных»

### 2. ТЕЛО
Развитие темы. Факты, история, эмоция. **Без воды.**
- Предложения ≤ 25 слов
- Конкретные цифры вместо «много»/«мало»
- Нет запрещённых слов (см. ниже)

### 3. ПРОДАЮЩАЯ ЧАСТЬ А→Б
Точка А (где читатель сейчас) → точка Б (где будет после действия). Конкретное CTA: ссылка, +DM, сохрани.

**Для Reels до 15 секунд:** в видео только хук. Продающая часть — в описании под видео.

---

## Продающая часть А→Б (обязательно в КАЖДОМ посте)

Формула: «Ты сейчас [точка А]. После [действие] ты [точка Б].»

| Рубрика | Точка А | Точка Б | CTA |
|---------|---------|---------|-----|
| Группа недели | не знаешь кого слушать | нашёл свою новую любимую группу | «Билеты: [ссылка]» |
| FOMO / Таймер | думаешь, успеешь | понял что осталось 100 билетов | «Не тяни: [ссылка]» |
| За кулисами | думаешь это обычный клубник | увидел как собирается сцена | «Хочешь быть там? [ссылка]» |
| Extreme Opros | не знал как думают музыканты | узнал неожиданный ответ | «А у тебя как? 👇» |
| Скетч Оли | скучно | улыбнулся | сохрани / репост |
| Финальный отсчёт | потом | сейчас | «[ссылка TicketsCloud]» |

## Каналы и форматы (адаптация одного поста)

| Канал | Длина | Ссылки | Формат | Особенности |
|-------|-------|--------|--------|-------------|
| **ВК** | 500–1500 зн. | Прямые ✅ | Пост / Карусель / VK Video | Длинные посты OK, фото обязательно |
| **ТГ** | 200–500 зн. | Inline-кнопка ✅ | Пост / одно фото | Короче и суше чем ВК |
| **Insta** | 300–800 зн. | ❌ в посте (только bio/Stories) | Reels / Карусель | Визуал главный, текст — дополнение |
| **VK Video** | Описание + CTA | Прямые ✅ | Описание под видео | Заголовок решает клик |

**Правило:** один пост = один исходник. Адаптация под канал **НЕ равно** копипаста. Каждый канал получает версию под свою аудиторию и формат.

## 9 рубрик Эстрим Феста — шаблоны

### 1. ГРУППА НЕДЕЛИ (Пн, S-01, Привлечение)
```
ХУК:  [Группа]. [Год]. [Жанр]. [Конкретный факт — тур, альбом, число концертов]
ТЕЛО: 2–3 предложения конкретики. Без «легенды» и «уникального звука».
А→Б: не знал → нашёл свою группу
CTA:  «Живьём — 11 июля. Билеты: [ссылка]»
Механика: Серия 1/12
```

### 2. EXTREME OPROS (Вт, S-10, Прогрев)
```
ХУК:  Вопрос музыканту — неожиданный или провокационный
ТЕЛО: Ответ музыканта (цитата, максимально живо)
А→Б: не знал как думают → узнал
CTA:  «А у тебя как? 👇»
Механика: Опрос стикером
Формат: Карусель рисованная (Оля делает визуал — НЕ AI)
```

### 3. ЗА КУЛИСАМИ (Вт, S-02, Прогрев)
```
ХУК:  Что происходит прямо сейчас — живо, без постановки
ТЕЛО: Фото/видео без подготовки. Детали которых не видит зритель.
А→Б: не видел → заглянул за кулисы
CTA:  «Хочешь быть там? [ссылка]»
Механика: Закулисье
```

### 4. СКЕТЧ ОЛИ (Сб, S-04, Лояльность)
```
Рисованный сюжет из жизни металлистов — ТЗ для Оли.
БЕЗ AI-арта. Оля рисует руками.
А→Б: скучно → улыбнулся / поделился
CTA:  Сохрани / Покажи другу
Механика: Сохрани, Репост
```

### 5. ТАЙМЕР ПРОДАЖ (Пт, S-23, Продажа)
```
ХУК:  «Осталось N дней»
ТЕЛО: «Продано X из 1400. Цена сейчас — [Y] ₽. Потом дороже.»
А→Б: потом → сейчас
CTA:  TicketsCloud [ссылка]
Механика: Цифра-удивляет
```

### 6. НОВОСТИ (по событию, S-06, Привлечение)
```
Группа подтверждена = 2 предложения + билеты. Без пафоса.
Пример: «[Группа] — в лайнапе. Официально. 11 июля. [ссылка]»
```

### 7. ИНФОПОВОДЫ (по ситуации, S-09, Прогрев)
```
Реакция на событие — концерт конкурента, трек группы из лайнапа, новость из мира метала.
Не сравнивать публично со Скрежетом. Только про нас.
```

### 8. КОНТЕНТ ОТ ГРУПП (по готовности, S-08, Привлечение)
```
Репост или видеоприглашение от группы + 1–2 предложения контекста от нас.
Пример: «[Группа] записали кое-что специально для вас. 11 июля будет ещё лучше. [ссылка]»
```

### 9. ФИНАЛЬНЫЙ ОТСЧЁТ (последние 2 нед, ежедневно, S-12, Продажа)
```
«7 дней. [конкретная деталь феста]. [сколько билетов осталось]. [ссылка]»
Ежедневно. Минимум 4–5 разных форматов вариации — не один шаблон.
```

## Workflow проверки (3 варианта)

**(а) Пакетно:** ContentOps готовит посты на неделю до среды. Critic проверяет пакетом до пятницы.

**(б) Совместная сессия:** ContentOps + Critic 1 час в неделю, правки на месте.

**(в) Асинхронно через канбан:** ContentOps → «На проверке» → Critic двигает в «Готово» или «Не принято».

**Правила:**
- Максимум **1 итерация** правок. Если пост вернулся дважды — садимся вместе.
- **Автономный режим:** если Critic недоступен — публикуем с пометкой «Не проверено». Пропуск публикации хуже чем пост без проверки.

## Google Sheets — workflow при создании черновика

1. Открой лист «Контентная» — найди строку текущей даты
2. Заполни колонки вручную: **H** (Текст), **I** (Цель), **J** (Продающая часть), **K** (Механика), **L** (Формат)
3. Колонки **D–F** (S-ID, Этап, Маяк) — подтягиваются VLOOKUP, **НЕ ТРОГАТЬ**
4. Охват план → колонка **M**
5. Статус → колонка **Q** = `Черновик`
6. Формулы в **R+** — **НЕ ТРОГАТЬ**
7. Сохрани копию в `output/drafts/YYYY-MM-DD-[рубрика]-[канал].md`
8. Запусти синхронизацию: `node scripts/migrate-drafts-to-sheet.js`

## Запрещённые слова (AI-маркеры)

Никогда не использовать:
`уникальный`, `качественный`, `индивидуальный подход`, `профессиональный`,
`позволяет`, `является`, `данный`, `предоставляет`, `обеспечивает`,
`в рамках`, `на сегодняшний день`, `в целях`, `осуществляется`,
`незабываемый`, `грандиозный`, `мероприятие`

---

## Gotchas

- Каждый пост = **одна идея**. Не пытаться запихать всё в один пост
- Первые 3 строки = хук. Аудитория решает листать дальше за **2 секунды**
- Контент-план готовится на **7 дней вперёд**, не поштучно
- «Нейросетей здесь не было» — указывать там где уместно (скетчи Оли, ручной контент Давида)
- AI-арт/AI-изображения — **НИКОГДА**. Давид и Оля рисуют сами
- Обратный отсчёт (14 дней до феста): минимум **4–5 форматов вариации**, не один шаблон
- Master / Олег Грановский — только уважительно
