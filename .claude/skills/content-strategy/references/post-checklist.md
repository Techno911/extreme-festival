# Чеклист поста перед публикацией (12 пунктов)

> Проверять каждый черновик перед переводом в «На проверке».
> Все 12 пунктов = зелёный. Хотя бы один красный = доработать.

---

- [ ] **1. ХУК есть** — первые 3 строки содержат вопрос / шок-факт / цитату / противоречие. Не начинается с «Привет» или названия события.

- [ ] **2. А→Б есть** — в посте явно прописана точка А (где читатель сейчас) и точка Б (где будет после действия). Формула «Ты сейчас [А] → [действие] → [Б]» читается в тексте.

- [ ] **3. CTA есть** — конкретный призыв к действию в конце: ссылка / «+DM» / «сохрани» / «напиши в комментах». Не «узнайте подробнее».

- [ ] **4. S-ID указан** — в мета-блоке прописан S-ID из карты коммуникаций (S-01, S-02, … S-23). Не пропущен.

- [ ] **5. Канал адаптирован** — длина и стиль соответствуют каналу:
  - ВК: 500–1500 зн., ссылка в тексте ✅
  - ТГ: 200–500 зн., короче и суше ✅
  - Insta: 300–800 зн., ссылок в посте НЕТ ✅
  - VK Video: описание + заголовок ✅

- [ ] **6. Запрещённые слова отсутствуют** — нет: `уникальный`, `качественный`, `позволяет`, `является`, `данный`, `предоставляет`, `обеспечивает`, `в рамках`, `на сегодняшний день`, `незабываемый`, `грандиозный`, `мероприятие`.

- [ ] **7. AI-текст отсутствует** — нет признаков машинного текста:
  - Предложения ≤ 25 слов
  - Есть конкретные цифры (не «много групп», а «12 групп»)
  - Есть живая интонация, не официальный тон
  - Нет перечислений в стиле «во-первых, во-вторых, в-третьих»

- [ ] **8. Длина в лимите** — текст не обрезается платформой. ВК: ≤1500 зн. ТГ: ≤500 зн. Insta: ≤800 зн.

- [ ] **9. Визуал указан** — прописано что нужно (фото / скетч Оли / видео с репетиции / карусель). Если «AI-арт» — СТОП, вернуть на доработку.

- [ ] **10. Ссылка рабочая** — если есть ссылка на билеты или сайт, она указана корректно. Для ТГ — через inline-кнопку, не голым текстом.

- [ ] **11. Статус в Sheets обновлён** — строка в листе «Контентная» заполнена (колонки H–M), статус Q = `Черновик`.

- [ ] **12. Копия в drafts/** — файл сохранён как `output/drafts/YYYY-MM-DD-[рубрика]-[канал].md`. Имя файла без кириллицы и пробелов.

---

## Быстрая проверка (если нет времени на все 12)

Минимальный контроль — **5 обязательных:**

1. ХУК не «Привет» и не название события
2. А→Б читается
3. CTA конкретный
4. S-ID есть
5. Нет AI-слов и нет AI-арта

Если все 5 ✅ — можно передавать на проверку Critic. Остальные 7 проверяет Critic.
