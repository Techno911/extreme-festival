Черновики постов хранятся в `/Users/techno/Desktop/Extreme_festival/output/content/`.

Файлы именуются: `posts_week_NN_MMMDD-DD.md` (например `posts_week_14_apr3-9.md`).

Выполненные недели:
- Неделя 14 (3–9 апреля 2026) — 5 постов готовы, 1 в блокере (скетч Оли)

S-ID формат: `VK-W{неделя}-{порядковый номер}`, например `VK-W14-001`.
Для ТГ-адаптаций: суффикс `-TG`, например `VK-W14-001-TG`.

**Why:** Единый формат нужен для отслеживания в Google Sheets и для Critic при проверке.
**How to apply:** При создании новых черновиков продолжать нумерацию недели. Не изобретать новые форматы S-ID без согласования.
