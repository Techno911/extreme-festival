Аудит проведён 2026-03-31. Все три раздела — статус ЧЕРНОВИК, прошли валидацию.

**Раздел А — Стратегия: ⚠️ Нужны правки**
- Кассовый разрыв не описан (бюджет 2,1 млн, выручка сейчас ~50к — откуда деньги)
- Конверсия сайта 3-5% завышена, реалистично 1,5-2,5%
- Instagram не помечен как «только брендинг, без конверсии»
- Метрику «охват посевов» заменить на «переходы на сайт»
- Нет пикселя ВКонтакте в требованиях

**Раздел Б — Сайт: ⚠️ Нужны правки**
- Super VIP «фото с артистами» не согласовано с группами — риск
- Критерий «3-30 человек» убрать, заменить на «кейс с zero-блоками»
- Нужна оговорка: лайнап поступает частями, сайт обновляется без подрядчика
- UTM-метки для TicketsCloud виджета не упомянуты

**Раздел В — Трейлер: ⚠️ Нужны правки**
- Сцена «рассвет» — съёмка в 5-6 утра, лишняя логистика
- Сцена «слэм» нереализуема — фест в июле, съёмка в мае. Нужны архивные кадры с группами (согласовать права)
- Нет плана дистрибуции трейлера на день запуска (8 июня)
- Согласование музыки с группой — нет дедлайна. Рекомендован дедлайн: 15 апреля
- Съёмку перенести на первую половину мая (не конец)
