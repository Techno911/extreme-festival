## Общее примечание по холдингу

Rock FM и НАШЕ Радио — входят в один холдинг, физически находятся по одному адресу (ул. Народного Ополчения, 39). Один PR-питч можно адресовать в оба направления одновременно через один контакт. Уточнить наличие единого PR-менеджера холдинга.

## Приоритет для Эстрим Феста

1. **НАШЕ Радио** — есть pr@nashe.ru, есть рубрика Афиша → питчить первым
2. **Rock FM** — метал/хард-рок формат → питчить вторым, через info@rockfm.ru
3. **Радио Максимум** — опыт с металом (Rammstein, BMTH), но поп-рок формат → третий
4. **Зайцев FM** — онлайн-аудитория, низкий приоритет для метал-феста
